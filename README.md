# Лабораторная работа №11 Таблица умножения

**Цель работы:** закрепить на практике знания по написанию программного кода для задач на языке С++.

**Ход работы:**

    #include <iostream>

    using namespace std;

    int main () {
       for (int i=1; i<=9; ++i){
         for (int j=1; j<=9; ++j){
            cout << i << "*" << j << "=" << i*j << endl;
        }
      }
    return 0;
    }
